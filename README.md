# Nawa Python Compiler

A simple Python compiler/interpreter that can execute Python code files or run in interactive mode.

## Features

- ✅ Compile and execute Python files
- ✅ Interactive REPL mode
- ✅ Syntax error detection with line numbers
- ✅ Runtime error handling with traceback
- ✅ Works in Termux and standard terminals
